## 📌 Project Overview

This project performs an end-to-end **Sales Performance Analysis** on a retail dataset (UCI Online Retail Dataset — 541,909 transactions). The goal is to uncover revenue trends, identify top-performing products and regions, and generate actionable business insights that a retail manager can act on.

**Business Problem:** A UK-based online retailer wants to understand:
- Which months and days generate the most revenue?
- Which products drive the most sales?
- Which countries contribute the highest revenue?
- Are there any cancellation patterns affecting revenue?

## 🔍 Key Insights

1. **📈 Revenue Peak in November** — Sales spike by ~40% in Nov, driven by holiday pre-orders. Retailers should ramp up inventory 4–6 weeks before.
2. **🏆 Top 10 Products = 18% of Revenue** — A small product set drives disproportionate revenue; priority restocking is essential.
3. **🌍 UK = 84% of Revenue** — Massive home-market dominance; international expansion to Germany and France shows early promise.
4. **❌ Cancellation Rate ~16%** — High cancellations on bulk orders suggest a need for order verification workflows.
5. **📅 Thursday is Peak Sales Day** — Marketing campaigns and promotions scheduled on Thursday–Friday yield the highest conversion.

## ▶️ How to Run

### 1. Clone the Repository
```bash
git clone https://github.com/prathap117/retail-sales-analysis.git
cd retail-sales-analysis
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Download the Dataset
Download `Online Retail.xlsx` from [Kaggle](https://www.kaggle.com/datasets/lakshmi25npathi/online-retail-dataset) and place it in the `data/` folder. Then convert to CSV or update the script path.

### 4. Run Data Cleaning
```bash
python scripts/data_cleaning.py
```

### 5. Run EDA & Generate Charts
```bash
python scripts/eda_analysis.py
```

### 6. Open Jupyter Notebook (Optional)
```bash
jupyter notebook notebooks/sales_analysis.ipynb
```

## 💼 Business Recommendations

- **Seasonal Promotions:** Launch email campaigns in October to capture the November revenue surge early.
- **Product Bundling:** Bundle top 10 SKUs with slow-moving products to improve overall basket size.
- **Cancellation Reduction:** Introduce order confirmation steps for orders above £500 to reduce the 16% cancellation rate.
- **Geographic Expansion:** Germany and Netherlands show consistent growth — target these markets with localised promotions.
